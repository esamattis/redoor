# Plan: Use Agent Name as ID and Handle Duplicate Connections

## Overview
Modify the agent connection logic to:
1. Use agent name directly as the agent ID (instead of generating unique ID with UUID)
2. Detect duplicate agent names on server-side
3. Exit agent process with non-zero exit code if duplicate detected
4. Add vitest test to verify duplicate agent name detection

**File**: `redoor-ui/tests/agents.test.ts`

Add `waitForExit` method to `ProcessManager` class (after line 47):
```typescript
async waitForExit(pid: number, timeoutMs: number = 10000): Promise<number | null> {
  const process = this.processes.get(pid)
  if (!process) {
    throw new Error(`Process not found: ${pid}`)
  }

  return new Promise((resolve, reject) => {
    const timeout = setTimeout(() => {
      reject(new Error(`Timeout waiting for process ${pid} to exit`))
    }, timeoutMs)

    process.once('exit', (code) => {
      clearTimeout(timeout)
      resolve(code)
    })
  })
}
```

```typescript
it('should reject duplicate agent names', async () => {
  const DUPLICATE_AGENT_NAME = 'duplicate-test-agent'

  // Connect first agent
  const firstAgentPid = processManager.spawn(AGENT_PATH, [WS_URL, DUPLICATE_AGENT_NAME])
  const firstAgent = processManager.getProcess(firstAgentPid)
  expect(firstAgent).toBeDefined()

  // Wait for first agent registration
  await waitForLogMessage(processManager.getProcess(serverPid)!, /Agent registered: agent_id=duplicate-test-agent/)

  // Verify first agent is listed
  const agentsAfterFirst = await apiClient.listAgents()
  expect(agentsAfterFirst.agents.some(a => a.name === DUPLICATE_AGENT_NAME)).toBe(true)

  // Try to connect second agent with same name
  const secondAgentPid = processManager.spawn(AGENT_PATH, [WS_URL, DUPLICATE_AGENT_NAME])
  const secondAgent = processManager.getProcess(secondAgentPid)
  expect(secondAgent).toBeDefined()

  // Wait for second agent to exit with error
  const exitCode = await processManager.waitForExit(secondAgentPid)
  expect(exitCode).not.toBe(0) // Non-zero exit code

  // Wait a bit for server to process the rejection
  await new Promise(resolve => setTimeout(resolve, 100))

  // Verify first agent is still the only one with this name
  const agentsAfterSecond = await apiClient.listAgents()
  const duplicateAgents = agentsAfterSecond.agents.filter(a => a.name === DUPLICATE_AGENT_NAME)
  expect(duplicateAgents.length).toBe(1)
})
```
