# Implementation Plan: `/api/v1/agents/{agent}` REST API

## Overview
Add REST API endpoint to retrieve detailed information about a connected agent, including system metrics and connection details.

## Files to Modify

1. `Cargo.toml` - Add sysinfo dependency
2. `src/commands.rs` - Add AgentInfo command, result, and REST response
3. `src/types.rs` - Update AgentRegister message with static system info
4. `src/actors/router.rs` - Update AgentInfo struct and add GetAgentDetails message
5. `src/main.rs` - Add REST API route and handler
6. `src/bin/redoor-agent.rs` - Send system info during registration

```rust
async fn agent_info(&self) -> CommandResult {
    use sysinfo::{System, SystemExt};
    use std::env;

    let pid = std::process::id();
    let cwd = env::current_dir()
        .map(|p| p.to_string_lossy().to_string())
        .unwrap_or_else(|_| "unknown".to_string());

    let mut sys = System::new_all();
    sys.refresh_all();
    let load_average = sys.load_average();
    let system_uptime = sys.uptime();

    CommandResult::AgentInfo(AgentInfoResult {
        pid,
        cwd,
        load_average,
        system_uptime,
    })
}
```

**Update execute method to handle AgentInfo:**

**Add GetAgentDetails handler:**
```rust
RouterMsg::GetAgentDetails { agent_id, reply } => {
    if let Some(agent_info) = state.agents.get(&agent_id) {
        state.rest_pending_responses.insert(agent_id.clone(), reply);
        let _ = agent_info.session_ref.cast(SessionMsg::OutgoingMessage(
            Message::Command {
                agent_id: agent_id.clone(),
                command: Command::AgentInfo,
            },
        ));
    } else {
        let error_msg = crate::commands::ErrorResponse {
            error: format!("Agent not found: {}", agent_id),
        };
        // Send empty response for agent not found
        let response = crate::commands::AgentDetailsResponse {
            id: agent_id.clone(),
            name: String::new(),
            pid: 0,
            cwd: String::new(),
            load_average_one: 0.0,
            load_average_five: 0.0,
            load_average_fifteen: 0.0,
            system_uptime: 0,
            os: String::new(),
            arch: String::new(),
            hostname: String::new(),
            username: String::new(),
            connected_at: 0,
        };
        let _ = reply.send(response);
    }
}
```

**Update RouteResponse handler to build AgentDetailsResponse:**
```rust
RouterMsg::RouteResponse { agent_id, result } => {
    let mut found = false;

    if let Some(reply) = state.rest_pending_responses.remove(&agent_id) {
        // Check if this is a GetAgentDetails response
        if let (Some(agent_info), CommandResult::AgentInfo(info_result)) =
            (state.agents.get(&agent_id), &result)
        {
            let response = crate::commands::AgentDetailsResponse {
                id: agent_id.clone(),
                name: agent_info.agent_name.clone(),
                pid: info_result.pid,
                cwd: info_result.cwd.clone(),
                load_average_one: info_result.load_average.0,
                load_average_five: info_result.load_average.1,
                load_average_fifteen: info_result.load_average.2,
                system_uptime: info_result.system_uptime,
                os: agent_info.os.clone(),
                arch: agent_info.arch.clone(),
                hostname: agent_info.hostname.clone(),
                username: agent_info.username.clone(),
                connected_at: agent_info.connected_at,
            };
            let _ = reply.send(response);
            found = true;
        } else {
            // Standard command response
            let _ = reply.send(result);
            found = true;
        }
    }

    if let Some(web_client_ref) = state.pending_responses.remove(&agent_id) {
        let _ = web_client_ref.cast(SessionMsg::OutgoingMessage(
            Message::CommandResponse {
                agent_id: agent_id.clone(),
                result: result.clone(),
            },
        ));
        found = true;
    }

    if !found {
        log!(Level::Warning, "No pending response found for agent_id={}", agent_id);
    }
}
```

**Add get_agent_details_handler function:**
```rust
async fn get_agent_details_handler(
    Path(agent): Path<String>,
    AxumState(state): AxumState<ServerState>,
) -> impl IntoResponse {
    match call_t!(
        &state.router_ref,
        |reply| actors::router::RouterMsg::GetAgentDetails { agent_id: agent.clone(), reply },
        30000
    ) {
        Ok(details) => {
            if details.name.is_empty() {
                // Agent not found (empty name indicates error)
                (
                    StatusCode::NOT_FOUND,
                    Json(ErrorResponse {
                        error: format!("Agent not found: {}", agent),
                    }),
                )
                    .into_response()
            } else {
                (StatusCode::OK, Json(details)).into_response()
            }
        }
        Err(e) => {
            let error_msg = format!("Failed to get agent details: {:?}", e);
            (
                StatusCode::INTERNAL_SERVER_ERROR,
                Json(ErrorResponse { error: error_msg }),
            )
                .into_response()
        }
    }
}
```

## Additional Details Returned

The API returns these easy-to-gather system details:

**Required:**
- name ✓
- pid ✓
- load_average (1, 5, 15 min averages) ✓
- cwd ✓
- system_uptime ✓
- connected_at ✓

**Additional:**
- os - Operating system name (linux, macos, windows)
- arch - CPU architecture (x86_64, aarch64)
- hostname - System hostname
- username - Current user running the agent
